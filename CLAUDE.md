# Project Violet — Claude Code Instructions

## Docs Workflow

Implementation plans live in `docs/` organized by status:

- `docs/upcoming/` — Ideas and plans not yet started
- `docs/doing/` — Actively being implemented
- `docs/done/` — Fully implemented

**After finishing implementation of a plan**, move its doc from `doing/` to `done/`.
**When starting work on an upcoming plan**, move it from `upcoming/` to `doing/`.
